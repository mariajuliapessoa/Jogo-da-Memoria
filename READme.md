# Jogo da Memória em JavaScript

## Descrição

Este projeto consiste no desenvolvimento de um jogo da memória para dois jogadores, utilizando HTML, CSS e JavaScript puro. O sistema foi construído com foco em manipulação do DOM, programação orientada a objetos e interatividade em tempo real.

O jogo permite que dois usuários insiram seus nomes, joguem alternadamente e acumulem pontos ao encontrar pares corretos de cartas.

---

## Funcionalidades

* Entrada de nomes dos jogadores
* Persistência dos dados utilizando LocalStorage
* Tabuleiro dinâmico criado via JavaScript
* Sistema de turnos entre dois jogadores
* Regra de jogo:

  * Acertou: continua jogando
  * Errou: troca de turno
* Controle de até duas cartas viradas por vez
* Verificação automática de pares
* Sistema de pontuação em tempo real
* Destaque visual do jogador atual
* Histórico dinâmico de jogadas
* Animação de flip nas cartas utilizando CSS
* Detecção de fim de jogo
* Exibição do vencedor
* Botão de reinício com reset completo do jogo

---

## Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox e animações 3D)
* JavaScript (ES6+)

## Arquitetura

O projeto segue uma separação clara de responsabilidades:

* HTML: estrutura das páginas
* CSS: estilização e animações
* JavaScript:

  * main.js: controle da tela inicial
  * jogo.js: manipulação do DOM e eventos
  * classes.js: regras de negócio e modelagem (Carta, Jogador, JogoMemoria)
